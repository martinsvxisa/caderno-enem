Configure seu computador para programar. Eu fiz um script para automatizar o criar e ativar o ambiente virtual, instalar a biblioteca pdf2image e conferir o seu repositório.

Se você está usando Linux, rode o arquivo linux-requirements.sh. O comando para rodar o arquivo é . linux-requirements.sh

Se você está usando Windows, rode o arquivo windows-requirements.sh. Para executar o arquivo no Windows, é só chamar o arquivo no terminal com windows-requirements.sh

Se você está usando Windows, adicionar os binários do poppler ao PATH
